# FringeheadAdventure
This is an indie-style game demo created entirely with Java.

![image](https://github.com/MiriamJB/FringeheadAdventure/assets/113400819/434338b8-e032-43b0-95db-b54fc6a61fd8)


## How to play:
**A**, **W**, **S** & **D**: use these keys to move.

![image](https://github.com/MiriamJB/FringeheadAdventure/assets/113400819/8ace77f8-627f-4a1c-a293-fc45c20ce3e2)


**C**: opens and closes the character screen. Use the **A**, **W**, **S** & **D** keys to scroll to different items, and press **ENTER** to equip/use those items.

![image](https://github.com/MiriamJB/FringeheadAdventure/assets/113400819/8ddc7d8f-256e-4b7e-b36b-82823aa2c426)


**F**: throws a starfish in the direction the player is facing.

![image](https://github.com/MiriamJB/FringeheadAdventure/assets/113400819/5272fc27-b2e1-4874-9eaa-925259f8b343)


**ENTER**: used to speak with NPCs and use the currently equipped weapon.

![image](https://github.com/MiriamJB/FringeheadAdventure/assets/113400819/54007121-a0b7-4f9d-bc95-62d10f1cdb6b)
